# Simple Todo App with Webflux and Kotlin

### APIs
CREATE Todo 
-   <b> POST </b> /todos

READ Todo
-   <b> GET </b> /todos/{id}

READ ALL Todo
-   <b> GET </b> /todos/

UPDATE Todo Status

-   <i>status: TODO, DOING, DONE </i>

-   <b> PUT </b> /todos/{id}/{status}

DELETE Todo
-   <b> DELETE </b> /todos/{id}
